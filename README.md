# Pokémon TCG Deck Builder Demo

**A Unity demo showcasing a card deck builder using the public Pokémon TCG API.**

---

## Overview

This project is a **technical demo** that demonstrates creating a **deck builder** for Pokémon Trading Card Game cards by fetching data from the **public Pokémon TCG API**.

Key features include:

* **Card selection and sorting** systems
* **Efficient data caching** to minimize API calls
* **Dynamic card loading** from an external API
* Clean, modular, and maintainable code suitable for learning or prototyping

This demo is intended as a showcase of **API integration, dynamic data handling, and UI systems in Unity**.

---

## Tech Stack

* **Engine:** Unity
* **Language:** C#
* **API:** Pokémon TCG API
* **Platform:** Windows (PC), WebGL

---

## Demo Goals

* Demonstrate **external API integration** in Unity.
* Showcase **dynamic content loading** and **data caching**.
* Highlight **interactive UI systems** for card selection and sorting.
* Provide a foundation for more advanced card game development or prototyping.
